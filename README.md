# [Net Worth Dashboard](https://bradguthrie-swa.github.io/networth-dashboard/)

Personal dashboard for displaying net worth built with React and Tailwind CSS. Demonstrates modern frontend workflows, API integration, and developer productivity with Cursor and AI-assisted development.

This project is configured for automatic deployment to GitHub Pages when code is pushed to the `main` branch.

It is available to view publicly at the following URL: **<https://bradguthrie-swa.github.io/networth-dashboard/>**

## Application Features

- Real time cryptocurrency price updates via CoinGecko API
- Visual allocation breakdown with interactive pie chart using Recharts library
- Account balance tracking and sorting
- Automatic price refresh with caching to prevent API rate limits

## Technology Stack

- React 19
- Vite
- Tailwind CSS
- Recharts
- React Context
- React Router DOM
- ESLint and Prettier
- Automated CI/CD with GitHub Pages for deployments

## Developer Setup and Prerequisites

This project requires the following technologies:

- Node.js 20 or higher
- npm
- git

To setup and run this application locally, run the following commands:

1. Install dependencies

   ```bash
   npm install
   ```

2. Start development server

   ```bash
   npm run dev
   ```
